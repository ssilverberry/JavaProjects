* Для успішної роботи середовища необхідно встановити JDK версії не нижче 5 та створити
    або оновити змінну середовища JAVA_HOME, в якій вказати повний шлях до директорії з JDK.
    Зробити це можна через панель керування, наприклад через пошук за словами environment variables.

* Для того, щоб знайти проект у дереві файлів зліва можна використовувати пункт 
    контекстного меню "Synchronize tree with current document"

* Для компіляції та тестування проекту можна використовувати команди меню 
    Macro > Зібрати Java проект, та інші.
